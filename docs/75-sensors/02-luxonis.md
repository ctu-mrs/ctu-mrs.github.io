# Luxonis

## Installation

The Luxonis driver on ROS2 Jazzy is supported by DepthAI v2, it can be installed by running
```bash
sudo apt install ros-jazzy-depthai-ros
```

## Startup

```bash
ros2 launch depthai_ros_driver camera.launch.py
```
